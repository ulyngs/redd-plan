# ReDD Map

A minimalistic, cross-platform Electron calendar app featuring the traditional Danish 6-month columnar calendar view.

## Features

- **Danish-style 6-month view**: See half a year at a glance in 6 columns
- **Click-to-edit dates**: Click any date to add notes or reminders
- **Automatic persistence**: Notes are saved automatically to localStorage
- **Year navigation**: Browse through years with arrow keys or buttons
- **Half-year toggle**: Switch between January-June and July-December
- **Dark mode**: Toggle between light and dark themes
- **Cross-platform**: Works on macOS, Windows, and Linux

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Run normally
npm start
```

## Building

```bash
# Build for current platform
npm run build

# Build for specific platforms
npm run build:mac
npm run build:win
npm run build:linux
```
